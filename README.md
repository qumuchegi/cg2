# cg2
cg2是 小跳蚤app 的服务器
 
# 功能：

  [用于服务‘小跳蚤’react app](https://github.com/qumuchegi/xiaotiaozaoapp)
  [小跳蚤](http://chegi.xyz)
     
# 技术栈：
     用 express-generator 生成程序框架
     node.js  +  express + mongodb
     
# 用到的模块：
   
     async（异步流程控制）
     body-parser（http请求体解析）
     mongoose （mongodb的node.js驱动)
